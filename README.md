🌐 Modern React Landing Page

A visually rich and responsive landing page built using React (Vite), Tailwind CSS, and shadcn/ui. This project is structured with reusable components and designed for a smooth, modern user experience.

🚀 Tech Stack
⚛️ React (Vite)
🎨 Tailwind CSS
🧩 shadcn/ui
🎞️ (Optional) Framer Motion for animations
📁 Project Structure
src/
│
├── components/
│   ├── HeroSection.jsx
│   ├── AboutSection.jsx
│   ├── MovementSection.jsx
│   ├── AchievementsSection.jsx
│   ├── TeamSection.jsx
│   ├── WhyTeamSection.jsx
│   ├── CTASection.jsx
│   └── Footer.jsx
│
├── pages/
│   └── Index.jsx
│
└── main.jsx
🧱 Page Composition

The main page (Index.jsx) includes the following sections:

🔹 HeroSection – Eye-catching landing section
🔹 AboutSection – Information about the project/company
🔹 MovementSection – Highlights mission or movement
🔹 AchievementsSection – Displays key accomplishments
🔹 TeamSection – Introduces team members
🔹 WhyTeamSection – Explains why to choose the team/product
🔹 CTASection – Call-to-action for users
🔹 Footer – Contact info and links
